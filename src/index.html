    ぱずぷれv3についてAbout PUZ-PRE v3

    　ぱずぷれv3はルールと図示された盤面から回答していくパズル、いわゆる"ペンシルパズル"を作成したり、遊んだりするWebアプリです。 現在は主にニコリに載っている/載っていたパズルを実装しています。 JavaScriptおよびHTML5の機能を使用して開発しています。
    PUZ-PRE v3 is a Web application that is for editing and playing pencil puzzles. Now almost all of supported puzzles are published by nikoli. I develop the script by JavaScript, and also work for implementing HTML5 feature.
    　スクリプトの作成は主にSafari9.0上で行っており、動作確認はその他のブラウザでも行っております。
    I develop the script mainly with Safari 9.0, test on other browsers.
